# Chinese (Simplified) locale content

This directory mirrors `docs-site/docs/` structure. Chapters are copied here
verbatim and translated in place. When `docs/` gets a new chapter, this
directory needs the same file (added by `npm run write-translations` where
possible, then hand-translated for docs content).

Plan 20 T6 does an initial pass on Part I and Part II. Remaining parts get
`TODO: 翻译` placeholders that still keep URL structure complete.
